# SweGBGv2

Landningssida byggd med Next.js 14 (App Router) + TypeScript + Tailwind CSS.

## Kom igång

```bash
npm install
npm run dev
```

Öppna http://localhost:3000

## Struktur

- `src/app/page.tsx` — all sidmarkup
- `src/app/layout.tsx` — fonter (Chakra Petch, IBM Plex Mono, Inter) + metadata
- `src/app/globals.css` — all styling
- `src/components/SiteEffects.tsx` — spotlight-cursor, terminal-typing, scroll-reveal
- `public/cyberjs.jpg` — bakgrundsbild i hero

## Bygg för produktion

```bash
npm run build
npm run start
```
